# 15. Appendices

### 15.1 Glossary

- **RAG:** Retrieval-Augmented Generation
- **LLM:** Large Language Model
- **ADK:** Agent Development Kit
- **BM25:** Best Matching 25 (keyword search algorithm)
- **Embedding:** Vector representation of text
- **Circuit Breaker:** Design pattern that prevents cascading failures by monitoring and controlling calls to external services
- **Stateless Agent:** An agent that does not maintain conversation history or memory between requests
- **Liveness Probe:** Health check that determines if an application is running and should be restarted if failing
- **Readiness Probe:** Health check that determines if an application is ready to accept traffic
- **Startup Probe:** Health check that determines if an application has completed initialization
- **Pydantic Settings:** Configuration management library that provides type-safe settings from environment variables
- **SecretStr:** Pydantic type that masks sensitive strings in logs and representations
- **Environment Variable:** System-level configuration value accessed by applications
- **Central Configuration:** Single source of truth pattern for application configuration
- **Configuration Validation:** Process of verifying configuration parameters at application startup
